01 // Mathematics

The Three Core Formulas

ACO is governed by three elegant equations that encode path selection probability, pheromone decay, and pheromone reinforcement.

EQ 01

Transition Probability

Pijk = [τij]α · [ηij]β Σl∈Nik [τil]α · [ηil]β
Intuition: At each junction, ant k picks the next city probabilistically — edges with more pheromone (τ) AND shorter distance (η) are exponentially more likely to be chosen. α and β control the trade-off.

This formula gives the probability that ant k, currently at node i, will move to node j next. It is a normalised product of two factors:

PkijTransition probabilityChance ant k moves from i→j
τijPheromone levelAccumulated chemical trail on edge i→j
ηijHeuristic (visibility)Typically 1/dij — closer nodes are more attractive
αPheromone weightHow much ants favour pheromone (>0); higher = more exploitation
βVisibility weightHow much ants favour short edges; higher = more greedy
NkiFeasible neighboursNodes ant k hasn't yet visited from position i
EQ 02

Pheromone Evaporation

τij ← (1 − ρ) τij
Intuition: Every iteration, ALL pheromone trails are multiplied by (1−ρ), shrinking them towards zero. Without new deposits, a trail eventually disappears — preventing the algorithm from getting permanently stuck.

Before new pheromone is added, every existing trail decays by a factor of (1−ρ). This global evaporation step is crucial for the algorithm's effectiveness.

SymbolNameMeaning
τijPheromone levelCurrent strength of trail on edge i→j
ρEvaporation rateFraction of pheromone lost per iteration (0 < ρ < 1)
1−ρRetention factorFraction of pheromone that survives
Why this matters: Without evaporation, paths discovered early would dominate forever. Evaporation implements "forgetting" — enabling the swarm to adapt when better routes appear.
EQ 03

Pheromone Reinforcement

τij ← τij + Σk=1m  Δτijk
where: Δτkij = Q Lk  if ant k used edge (i,j), else 0

After all ants complete their tours, each ant deposits pheromone proportional to the quality of its solution. Shorter/better paths receive more pheromone.

SymbolNameMeaning

QConstantScaling factor (controls pheromone quantity)

Key insight: Q/Lk means a tour half as long deposits twice as much pheromone. Quality is rewarded non-linearly, driving rapid convergence to better solutions.
02 // Algorithm

Step-by-Step: How ACO Runs

Each iteration of ACO consists of three phases repeated until convergence or a stopping criterion is met.

01

Initialise

Set τij = τ0 (small constant) on all edges. Place m ants at random start nodes. Set α, β, ρ, Q parameters.

02

Construct Solutions

Each ant builds a complete tour by repeatedly choosing the next node using the transition probability formula (EQ 01).

03

Evaluate Tours

Compute the total cost Lk for every ant's tour. Track the global best solution found across all iterations.

04

Evaporate

Apply τij ← (1−ρ)τij to ALL edges simultaneously (EQ 02). Weak paths fade towards zero.

05

Deposit Pheromone

Each ant deposits Δτkij = Q/Lk on every edge it traversed (EQ 03). Better tours → more deposit.

06

Repeat / Converge

Return to step 2. Iterate until max iterations reached, no improvement for N rounds, or solution quality threshold met.
